# Phishing URL Inspector

## Описание проекта
**Phishing URL Inspector** — это небольшой учебный проект, созданный для демонстрации проверки веб-ссылок (URL) на возможные признаки фишинга.  
Файл `example1.html` реализует простую клиентскую проверку и показывает, как можно анализировать ссылку на уровне фронтенда без серверной части.

Проект предназначен для учебных и исследовательских целей в области кибербезопасности.

---

## Основные возможности
- Ввод ссылки в форму и её проверка.  
- Простая логика оценки риска (`riskScore`) с помощью JavaScript.  
- Определение базовых признаков небезопасных ссылок (например, использование `http` вместо `https`).  
- Отображение результата проверки пользователю.
